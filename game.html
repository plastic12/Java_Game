Description

It is a shooting game that user have to collect color dots to enter the next level. User can also collects upgrades to fill the Power bar, which can upgrade the bullets to make them more powerful. There are a total of 7 levels. Users have to collect color dots to enter the next level.

There are 3 types of upgrades: black dots for filling the power bar, white dots for raising score, and other colors are for filling the progress bar.

Instructions

User arrow keys to move and mouse to control directions of bullets.
